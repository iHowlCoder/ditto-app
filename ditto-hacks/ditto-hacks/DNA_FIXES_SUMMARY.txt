TROUBLESHOOTING ORIENTATION
----------------------------------------------------------------

If model is STILL laying flat after +90° X:

Try these in DNAOrientationTester:

Option 1: Rotate on Y-axis
  rotationX = 0
  rotationY = 90
  rotationZ = 0

Option 2: Rotate on Z-axis
  rotationX = 0
  rotationY = 0
  rotationZ = 90

Option 3: Combined rotation
  rotationX = 90
  rotationY = 90
  rotationZ = 0

Option 4: Different X angle
  rotationX = -90  (try negative again)
  rotationY = 0
  rotationZ = 0

The bounding box output will tell you which dimension is largest:
- If Width (X) is largest → model is sideways
- If Height (Y) is largest → model is upright ✓
- If Depth (Z) is largest → model is laying forward/back
